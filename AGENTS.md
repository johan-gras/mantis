## Build & Run

Build: `cargo build --release`
Run CLI: `cargo run -- --help`

## Validation

Run these after implementing to get immediate feedback:

- Tests: `cargo test`
- Lint: `cargo clippy -- -D warnings`
- Format check: `cargo fmt --check`
- Format fix: `cargo fmt`
...
